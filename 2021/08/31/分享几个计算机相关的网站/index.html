分享几个计算机相关的网站

发表于 2021-08-31 分类于 其他 阅读次数：
之前收藏了几个有意思的网站，有空的时候可以逛逛，或者找计算机的一些资料也比较方便

大家好呀，作为程序员，分享几个我用过的网站，感觉还挺不错的。

sqlbolt

https://sqlbolt.com/

这是一个练习SQL语句的地方，小白也可以学习。

regexone

https://regexone.com/

这是一个练习正则的地方，跟第一个属于同一个作者开发。这种交互式的网站还挺有意思的。

Learn Git Branching

https://learngitbranching.js.org/?locale=zh_CN

这是一个练习Git的网站，非常实用。通过分支的可视化，可以帮助你更好理解你的每一步Git命令做了什么事情。

找计算机书籍

https://www.xz577.com/

这里的资源比较多，可以找到很多PDF类型的计算机书籍。

另一个我比较常用的是《微信读书》，里面的书籍也很多。

牛客网

https://www.nowcoder.com/

用来面试刷题的，我觉得跟leetcode的一个差别是在牛客网可以刷一些计算机基础的题目，比起leetcode纯粹都是算法题会更广一些。

iconfont

https://www.iconfont.cn/

这个主要用来找各种Icon的，比较适合在学习做项目又缺少设计同学的情况。
